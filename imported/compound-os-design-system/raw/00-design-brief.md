# Compound OS Product Requirements Document

## Vision
Compound OS is a $49 founding-membership digital system by Tarek Shaar, delivered as a tightly-produced library of modules across three pillars: Markets, Fitness, and Mindset. It is designed for ambitious operators (25–45) who value technical precision and editorial gravitas.

## Brand DNA
- **Tagline:** "the operating system for a compounding life."
- **Brand Mark:** Three-bar Fibonacci stack (5:8:13 ratio) in champagne #BF9A62.
- **Palette:** Dark-first. Background #0a0b0f, Text #f4f4f5.
- **Pillar Accents:**
  - Markets/Trading: #00d4aa (Teal)
  - Fitness: #f97316 (Orange)
  - Mindset: #a78bfa (Lavender)
- **Core Accent:** Champagne #BF9A62 (Luxury only).

## Design Directions

### 1. Editorial Quarterly
- **Typography:** GT Sectra (Display Serif) + Söhne/Inter Tight (Precision Sans).
- **Vibe:** Magazine vertical rhythm, oversized pull quotes, hairline rules, generous whitespace.
- **Imagery:** B&W grain photography, negative space.
- **Motion:** Cinematic.

### 2. Private Terminal
- **Typography:** Söhne / Inter Tight / PP Neue Montreal (Sharp Sans).
- **Vibe:** Tight grid, monospace numerals, data-viz motifs (sparklines, candle silhouettes).
- **Imagery:** Functional, technical textures.
- **Motion:** Snappy.

### 3. Quiet Luxury
- **Typography:** Neue Haas Grotesk / ABC Diatype (Editorial Sans).
- **Vibe:** Hairline borders, gold-foil hovers, material imagery (concrete, brushed metal).
- **Imagery:** Textural, material-focused.
- **Motion:** Restrained.

## Core Pages & Components
- **Pages:** Home, Pricing, Login, Stripe Success, Onboarding, Dashboard, Module Page, Trust Pages.
- **Components:** Fibonacci Navigation, Pillar Cards, Module Cards, Stat Tiles, Auth Cards, Long-form Prose Container.
